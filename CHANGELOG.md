# Changelog - Matchering 2025

All notable changes to this project will be documented in this file.

## [2025] - Major Refactor

### Added
- Multi-reference support (up to 10 reference tracks)
- Hot or Not comparison system with hover previews
- Real-time ranking charts
- WAV 16-bit and 24-bit export formats
- Drag and drop file upload interface
- Python 3.14+ compatibility
- Docker deployment scripts
- Local execution scripts for Windows

### Changed
- Complete UI redesign with hover-based previews
- Synchronized preview playback for fair comparison
- Automatic comparison continuation after voting
- Download filenames with random codes
- Removed advertisements and donation buttons
- Simplified navigation

### Fixed
- Preview synchronization issues
- Comparison algorithm to prevent duplicate masterings
- Python 3.14 compatibility issues
- Preview playback (WAV format)

### Removed
- MP3 export functionality (removed due to implementation issues)
- FFmpeg dependency (no longer needed)
- Donation buttons and links
- Advertisement integrations
- External analytics
- Unnecessary Docker troubleshooting scripts
